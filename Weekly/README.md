# Weekly

Weekly summaries that synthesize the week's activity.

**File naming:** `YYYY-WNN.md` (e.g., `2026-W18.md`)

**Typical content:**
- Key developments across coverage
- Thesis changes or new signals
- Portfolio-level observations
- What to watch next week
- Lint summary (run the wiki health check weekly)

**Tip:** Friday afternoon is the natural time to generate this. Ask Claude to summarize the week's log.md entries and flag anything that changed a thesis.
