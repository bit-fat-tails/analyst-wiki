# Hiring

Hiring materials and candidate evaluation.

**What goes here:**
- Job descriptions and requirements
- Interview rubrics and case studies
- Candidate evaluation notes
- Onboarding checklists for new team members

**Tip:** When onboarding a new analyst, point them at this wiki. The getting-started guide (`docs/getting-started.md`) and philosophy doc (`docs/philosophy.md`) explain the system. Having a new hire set up their coverage in the wiki is a great onboarding exercise.
