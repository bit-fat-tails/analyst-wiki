# Ideas

New investment ideas — the pipeline from signal to thesis.

**File naming:** `YYYY-MM-DD-{slug}.md` (e.g., `2026-05-01-long-tsm-cowos-bottleneck.md`)

**Typical content:**
- Thesis summary (2-3 sentences)
- The setup (what's happening)
- The insight (what we see that others don't)
- Key catalysts and timeline
- Risks and what proves us wrong
- Sizing rationale

**Lifecycle:** Ideas start here. If conviction builds, they graduate to a formal thesis document in `Companies/{TICKER}/{TICKER}-{thesis-slug}.md` using the thesis template.
