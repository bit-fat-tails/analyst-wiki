# Daily

Daily logs and briefings. One file per day.

**File naming:** `YYYY-MM-DD.md`

**Typical content:**
- Pre-market scan: overnight news, earnings, macro moves
- Intraday notes: trades, calls, observations
- End-of-day: what changed, what to follow up on

**Tip:** Start each morning by asking Claude to scan your coverage for overnight developments and write the daily log. This is your most natural ingest trigger.
