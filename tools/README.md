# Tools

Custom tooling that supports your research workflow.

**Examples of what might live here:**
- Alert ingestion scripts (email → markdown)
- Data fetching utilities (API wrappers for financial data providers)
- Search tooling (local markdown search, e.g., [qmd](https://github.com/tobi/qmd))
- Chart generation scripts
- Portfolio analytics

**Tip:** As the wiki grows past ~100 pages, you'll want a search tool beyond the index files. qmd provides hybrid BM25/vector search over markdown files with an MCP server, so Claude can use it natively. Start simple — you can always add tooling later.
